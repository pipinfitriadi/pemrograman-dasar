# Hitung Digit Ganjil dan Genap

Buat program `digit_stat.py` yang:

1. Membaca satu bilangan bulat positif **Y**.
2. Tanpa konversi string, gunakan **while** untuk mengekstrak digit dan hitung digit ganjil & genap.
3. Tampilkan hasil.
4. Contoh:
    ```log
    Y = 123456
    Digit Genap : 3
    Digit Ganjil: 3
    ```
