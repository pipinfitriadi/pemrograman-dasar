# Analisis String Sekuen

Buat program `analisis_string.py` yang:

1. Membaca sebuah kalimat (string).
2. Dengan **for**, hitung jumlah huruf vokal & konsonan. Abaikan karakter non-huruf.
3. Cetak:
    ```log
    Vokal     : <count>
    Konsonan  : <count>
    ```
4. Contoh:
    ```log
    Kalimat   : Halo, Dunia!
    Vokal     : 5
    Konsonan  : 4
    ```
